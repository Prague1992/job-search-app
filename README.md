## Available Scripts

In the project directory, you can run:

### `npm run start`

Runs the server and client in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
The server runs on the port 9000.

The page will reload if you make edits.<br />
You will also see any lint errors in the console.

### `npm run start-server`

Starts the server on the port 9000.

### `npm run start-front`

Starts the client on the port 3000.

In the Application, clicking on the header "devjobs" will route to the home page again.
